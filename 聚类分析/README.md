## 数据预处理：正向化[^1]+标准化[^2].
[^1]:转化为极大型的数据，让数字越大越好
[^2]:将原始各指标数据按比例缩放，去除数据的单位限制，将其转化为无量纲的纯数值，便于不同单位或量级的指标能够进行比较和加权。 数据标准化最典型的就是数据的归一化处理，即将数据统一映射到[0,1]区间上
